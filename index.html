Main Project
Stastical analysis on experimental mechanics

    Learn about tissue biomechanics by using statistics and data analysis method to design, analyse and critically interpret materials science data. It also teach about understanding of mechanical behaviour of materials. The main objectives of this project are engineering science in answering questions, study design and underlaying theorisation, critical analyses of results and coherent data presentation. The softwares used to analyze and clean the raw data are python using the Visual Studio Code and Microsoft Excel. The skills learned during this course are Matplotlib, GGplot, linear regression, Data vizualization using boxplot and t-test, noise filtering function in Excel and python, Numpy, Scipy signal and Pandas.
